## Architecture

```
ONEM (Founder)
  └── PROXY (Chief of Staff / System Orchestrator)
        ├── NARRYON  — Storytelling Engine
        ├── VISUYON  — Production Engine (image, video, voice)
        ├── VIRYON   — Growth Engine (reach, platforms)
        ├── CODEXYON — Pipeline Architect (automation, web, publish)
        └── FINYON   — Financial Intelligence Core
```

## Workflow

```
Idea (Onem) → Proxy validates → Narryon (narrative) → Visuyon (production)
→ Viryon (distribution strategy) → Codexyon (publish + automate)
→ Finyon (analyze) → Proxy (adjust strategy)
```

- **Project tracking:** GitHub Issues + Projects
- **Agent memory:** `registry/` directory (shared context)
